# 🍳 Lezzet Atlası

Yemek yapmak için vakti kısıtlı olan, elindeki malzemeler sınırlı olan ve aradığı tarifi hızlıca bulmak isteyen kişiler için tasarlanmış Türk yemek tarifleri platformu.

## 🌐 Canlı Site

[hirakarakuzu.github.io/lezzet-atlasi](https://hirakarakuzu.github.io/lezzet-atlasi)

## ✨ Özellikler

- 96+ özgün Türk yemek tarifi
- Elindeki malzemelere göre tarif arama
- Kategori bazlı filtreleme
- Favori tarif kaydetme
- Karanlık / Aydınlık mod
- Admin paneli — tarif ekleme, düzenleme, silme
- Kullanıcı giriş sistemi
- Mobil uyumlu tasarım

## 🛠️ Kullanılan Teknolojiler

- HTML5, CSS3, JavaScript (Vanilla)
- Google Fonts — Plus Jakarta Sans, Manrope
- Font Awesome — ikonlar
- Formspree — iletişim formu
- GitHub Pages — yayın
